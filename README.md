# 🦆 Open Ducky

An [open-source](./LICENSE) compiler and virtual machine for [DuckyScript 3.0](https://documentation.hak5.org/hak5-usb-rubber-ducky/duckyscript-tm-quick-reference) for [Linux](https://www.kernel.org/). Makes use of the [Linux USB HID gadget driver](https://www.kernel.org/doc/html/latest/usb/gadget_hid.html), allowing it to run on a multitude of devices.

## Officially Supported Hardware

Currently, this project is only tested on the [Raspberry Pi Zero W](https://www.raspberrypi.com/products/raspberry-pi-zero-w/).
